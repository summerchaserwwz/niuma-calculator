## 为什么要做这个

工资只是工作价值的一部分。同样月薪 2 万：

- 一个人每天通勤 2 小时 + 996 + 烂领导 → **实际性价比极低**
- 另一个人 5 分钟步行 + 弹性 + 好领导 + 铁饭碗 → **实际性价比极高**

这个计算器综合 **5 大维度**，给你一个客观的打分。

---

## 计算逻辑详解

### 🔢 核心公式

```
最终得分 = 薪资基础分 × 稳定性系数 × 氛围系数 × 自由度系数 × 通勤系数
```

> 100分 = 符合预期；>100 = 值；<75 = 值得重新评估

---

### 💰 维度一：薪资基础分（基础）

**计算步骤：**

1. **年总收入** = 月薪 × 12 + 年终奖（月薪×N）+ 月均补贴 × 12
2. **有效工作日** = 52周 × 每周工作天数 − 年假 − 法定假日 − 带薪病假 × 0.5
3. **日均有效耗时** = 日工时 + 单程通勤×2×(1 − WFH比例)
4. **等效时薪** = （年总收入 / 有效工作日）/ 日均有效耗时

**期望时薪** 基于三个参数：
- 行业基准时薪（互联网 ¥95/h，金融 ¥88/h，制造 ¥44/h…）
- 学历系数（专科 0.78 → 博士 1.35）
- 工龄期望（0年=×1.0，15年封顶=×1.41，正向增长）

```
薪资基础分 = 等效时薪 / 期望时薪 × 100
```

---

### 🏛️ 维度二：稳定性系数（影响幅度 ±35%）

工作稳定性 = 免去的「失业焦虑成本」+ 社保/退休/贷款的隐性价值

| 公司类型 | 系数 |
|---------|------|
| 公务员 | ×1.35 |
| 事业单位/编制 | ×1.28 |
| 央企/国企 | ×1.20 |
| 大外企/500强 | ×1.10 |
| 国内大厂 | ×1.05 |
| 中型公司（基准）| ×1.00 |
| 小公司 | ×0.90 |
| 初创/创业 | ×0.78 |

---

### 🌟 维度三：氛围系数（影响幅度 ±28%）

```
氛围系数 = 领导系数^0.7 × 同事系数^0.3
```

**领导质量**权重 70%（决定你的晋升、成长和每日情绪）：

| 评分 | 系数 | 对应描述 |
|------|------|--------|
| 1 | ×0.72 | PUA/甩锅/画大饼 |
| 2 | ×0.86 | 能力差或人品差 |
| 3 | ×1.00 | 正常 |
| 4 | ×1.13 | 支持下属、给资源 |
| 5 | ×1.24 | 导师型，跟对人 |

**同事氛围**权重 30%（影响日常体验）：
- 1分(×0.85) 内斗拆台 → 5分(×1.12) 互帮互助

---

### 🕊️ 维度四：自由度系数（影响幅度 ±35%）

```
自由度系数 = 加班系数 × (1 + WFH加成) × 压力/自主系数
```

**加班程度**（影响最大）：

| 加班强度 | 系数 |
|---------|------|
| 几乎不加班 | ×1.15 |
| 偶尔加班 | ×1.05 |
| 经常加班 | ×0.90 |
| 大量加班/大小周 | ×0.78 |
| 严重内卷/996 | ×0.65 |

**WFH 加成**：每周 ≥3 天远程 +8%，≥1 天 +4%

**压力/自主权**：1分(×0.88) 高压无自主 → 5分(×1.10) 低压高自主

---

### 🚇 维度五：通勤系数（影响幅度 ±29%）

有效通勤 = 单程时长 × (1 − WFH比例)，折减后计算

| 有效单程 | 系数 |
|---------|------|
| ≤15分钟 | ×1.14 |
| 15~30分钟 | ×1.06 |
| 30~45分钟（基准）| ×1.00 |
| 45~60分钟 | ×0.93 |
| 60~90分钟 | ×0.83 |
| >90分钟 | ×0.71 |

---

### 📊 "打败了多少人"的计算方式

基于各行业的**正态分布参数**（均值+标准差），使用标准正态 CDF 计算你的分数对应的百分位：

```javascript
// Abramowitz & Stegun 近似，误差 < 7.5e-8
percentile = normalCDF(finalScore, industryMean, industryStd)
beat = percentile × 100%
```

数据来源：参考中国 2024 年薪资调研报告中各行业分布估算。
